About This Example

This page demonstrates how to include CSS directly in an HTML document using the <style> tag in the head section.

The styling includes colors, fonts, spacing, shadows, and even some simple animations on hover.

Benefits of Embedded CSS

Embedding CSS directly in your HTML document is useful for:

  • Simple projects where you don't need separate files
  • Quick prototyping and testing
  • Single-page applications with limited styles
  • Ensuring styles load immediately with the page
